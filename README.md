UTP - MARKETPLACE

UTP Marketplace es una plataforma web desarrollada para la comunidad universitaria de la UTP, diseñada para facilitar el intercambio, compra y venta de materiales académicos, herramientas de estudio y servicios entre estudiantes.

CARACTERÍSTICAS PRINCIPALES

● Catálogo Dinámico: Filtrado de productos por categorías (Libros, Electrónica, Proyectos, etc.) mediante parámetros de URL.

● Carrito de Compras: Gestión persistente de productos mediante base de datos PostgreSQL, garantizando la integridad de los datos del usuario.

● Detalle de Producto: Vista dinámica que carga información específica del producto seleccionado.

● Interfaz Responsiva: Diseño moderno y limpio construido con Tailwind CSS.

● Navegación Intuitiva: Menús activos que indican la ubicación del usuario.

TECNOLOGÍAS UTILIZADAS

Frontend:

● HTML5: Estructura semántica del contenido.

● Tailwind CSS: Framework de estilos para un diseño moderno y rápido.

● JavaScript: Lógica de negocio, filtrado, manejo de carrito y manipulación del DOM (Document Object Model).


Backend:

● Java (Spring Boot): API RESTful que gestiona la lógica del lado del servidor.

● Maven: Gestión de dependencias del proyecto.

● Base de Datos: Gestión y administración de esquemas mediante pgAdmin utilizando PostgreSQL.
